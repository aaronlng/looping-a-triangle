# Looping-a-Triangle Exercise

Exercise found in Chapter 2 of [Eloquent JavaScript](https://eloquentjavascript.net/).

*Write a loop that makes seven calls to console.log to output the following triangle:*

    `#`  
    `##`  
    `###`  
    `####`  
    `#####`  
    `######`  
    `#######`
